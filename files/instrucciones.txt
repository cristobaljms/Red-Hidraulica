1) instalar los paquetes que se encuentran en el archivo de requirements.txt

    pip install -r requirements.txt

2) instalar RabbitQM en la pc

    La instalacion puede variar dependiendo de si es para windows o linux.

    Windows:
        https://www.rabbitmq.com/install-windows.html

    Linux:
        apt-get install rabbitmq-server

3) instalar celery

    se puede instalar de varias formas:

        desde aqui
            http://www.celeryproject.org/install/

        o usando pip
            pip install Celery

4) la base de datos se encuentra dentro de proyecto en la carpeta de files:
    usar postgres 9 o mayor

5) ejecutar el proyecto.
    Para modo desarrollador deben abrir 2 terminales, una donde correra celery y en la otra el django

    en una terminal nos ubicamos dentro de la carpeta, ya sea con el comando cd o DIR
     y ejecutamos django:
        python manage.py runserver

    y en la otra terminal ejecutamos el celery:
        celery -A root worker -l info
